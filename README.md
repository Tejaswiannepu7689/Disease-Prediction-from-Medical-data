# Disease-Prediction-from-Medical-data

## Objective
Predict diseases using patient medical data and Machine Learning algorithms.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Installation

pip install -r requirements.txt

## Run

python disease_prediction.py

## Features
- Data preprocessing
- Disease prediction
- Accuracy evaluation
- Performance visualization
- Multiple ML model comparison

## Dataset
Medical dataset containing patient health information.

## Output
- Disease prediction
- Accuracy score
- Classification report
- Graphical analysis
